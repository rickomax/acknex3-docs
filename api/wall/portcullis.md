# `PORTCULLIS` `(flag)`

Attaches the texture to the wall's upper/lower edge (lifts, trapdoors); aligned by POSITION (0 = bottom, 1 = top).

_Availability: WDL (1995 book) · ACKNEX v3.8 / v3.9._
