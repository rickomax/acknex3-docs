# `RULE` `Skill = expression;`

Assigns an arithmetic expression (numbers, skills, parentheses, + - * /) to a skill; clamped by MIN/MAX. Avoid factors < 0.01.

_Availability: WDL (1995 book) · ACKNEX v3.8 / v3.9._
