# `PLAY_FLIC` `Flic;`

Plays a full-screen animation. In the 1995 engine, image build and EACH_TICK stop during play (start a song one frame earlier); newer engines keep EACH_TICK running and expose FLIC_FRAME.

_Availability: WDL (1995 book) · ACKNEX v3.8 / v3.9._
