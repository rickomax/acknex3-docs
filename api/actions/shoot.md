# `SHOOT` `[Object];`

Triggers IF_HIT on the nearest object in view (using SHOOT_FAC/RANGE/X/Y); sets HIT_DIST and RESULT. Also opens nearby switches/doors.

_Availability: WDL (1995 book) · ACKNEX v3.8 / v3.9._
