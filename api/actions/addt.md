# `ADDT` `[Obj.]Key, Number/[Obj.]Key2;`

Like ADD but time-corrected (value multiplied by TIME_CORR). For constant-speed changes (e.g. lifts).

_Availability: WDL (1995 book) · ACKNEX v3.8 / v3.9._
