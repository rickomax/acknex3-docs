# `REFRESH` `(FLAGS REFRESH;)`

Regenerates the panel each frame (e.g. for scrolling text). Costly for large panels — prefer an OVERLAY.

_Availability: WDL (1995 book) · ACKNEX v3.8 / v3.9._
