# `BMAPS` `Bitmap1, Bitmap2, …;`

List of up to 64 bitmap keywords (count = `SIDES * CYCLES`). Front-side phases first, then further sides clockwise.

_Availability: WDL (1995 book) · ACKNEX v3.8 / v3.9._
