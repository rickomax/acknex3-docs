# `SIDES` `Number;`

Number of sides (1..64, default 1). The visible side changes with the viewing angle, giving Things/Actors a 3D appearance. Wall textures may have two sides.

_Availability: WDL (1995 book) · ACKNEX v3.8 / v3.9._
