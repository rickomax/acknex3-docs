# `PALFILE` `<Filename>;`

Graphics file whose colours are used for the base palette. Colours 0 and 1 carry the colour for infinite distance (black indoors / foggy white outdoors).

_Availability: WDL (1995 book) · ACKNEX v3.8 / v3.9._
