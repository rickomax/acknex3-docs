# `NEXUS` `Number;`

Sets the size of the Nexus — the internal data structure used for rendering. Depends on the maximum number of objects, walls and regions visible at once. Default 14. Too small a Nexus produces image errors signalled by the PC speaker.

_Availability: WDL (1995 book) · ACKNEX v3.8 / v3.9._
