# `CLIP_DIST` `Number;`

Global view distance in Steps. Walls and objects entirely beyond this distance from the player are not rendered. Default 1000. Can be overridden per region.

_Availability: WDL (1995 book) · ACKNEX v3.8 / v3.9._
