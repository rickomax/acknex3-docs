# `VIDEO` `Keyword;`

Sets the screen resolution. Predefined resolution keywords: `320x200`, `X320x240`, `X320x400`. Newer engine versions add SVGA modes `S640x480` and `S800x600`.

> **Note:** The 1995 book prints the first keyword as `X320x200`; the correct keyword is `320x200` (no prefix).

_Availability: WDL (1995 book) · ACKNEX v3.8 / v3.9._
