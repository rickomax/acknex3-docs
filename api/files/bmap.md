# `BMAP` `Keyword, <Filename> [, x, y, dx, dy];`

Assigns a bitmap (`.PCX`, `.LBM` or `.BBM`) to the keyword, or a rectangular section of it (upper-left corner `x,y`, size `dx,dy`, in pixels).

_Availability: WDL (1995 book) · ACKNEX v3.8 / v3.9._
