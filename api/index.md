# API Reference

WDL keywords grouped by object type (listed alphabetically). Pick a type, then a
keyword — or browse the flat [Keywords A–Z](../keywords.html) index.

| Object type | Keywords | Summary |
|:------------|:--------:|:--------|
| [Actions & Instructions](actions.html) | 48 | ACTION definitions, the instructions used inside them, and the global/event action keywords. |
| [Actors (ACTOR)](actor.html) | 42 | The ACTOR object: animate objects with movement, targets, events and flags. Things share these keywords. |
| [Files](files.html) | 6 | Keywords that assign external files (bitmaps, sounds, songs, animations) to a name. |
| [Globals & Predefined Skills](globals.html) | 98 | Skill definitions and the predefined player, engine, input, time and debug skills. |
| [Palettes](palette.html) | 4 | Definition of the 256-colour palette, shading ranges and colour animations. |
| [Panels, Texts & Overlays](ui.html) | 18 | The 2D user-interface objects: PANEL, TEXT and OVERLAY, their members, and the screen lists. |
| [Predefined Settings](settings.html) | 11 | Global engine settings, normally placed at the start of the main WDL file. |
| [Regions (REGION)](region.html) | 26 | The REGION object: floor/ceiling areas bounded by walls, with textures, lighting and events. |
| [Synonyms](synonyms.html) | 3 | Predefined names that let an action address whichever object/region triggered it. |
| [Textures (TEXTURE)](texture.html) | 18 | The TEXTURE object and its properties, flags, sound and animation parameters. |
| [Things (THING)](thing.html) | 1 | The THING object: simple inanimate objects sharing the ACTOR keyword set. |
| [Walls (WALL)](wall.html) | 31 | The WALL object: vertical surfaces separating regions, with textures, parameters, events and flags. |
| [Ways (WAY)](way.html) | 2 | The WAY object: lists of waypoints an Actor travels along. |
